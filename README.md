# Lexora

Lexora is an original multilingual word-connect crossword puzzle game.

## Current version

This repository currently contains the first production foundation:

- Vite + React + TypeScript
- English, Spanish, Russian and Turkish interface labels
- starter levels for all four languages
- crossword grid rendering
- letter wheel interaction with mouse/touch support
- main-word and bonus-word validation
- local progress persistence
- responsive premium visual shell
- theme-based background system

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Preview production build

```bash
npm run preview
```

## Project direction

Lexora is not a clone of any branded game. It uses the common word-connect crossword mechanic with original name, code, visuals, levels, dictionaries and progression.

Planned v1 target:

- 300 curated levels per language
- 4 languages: English, Spanish, Russian, Turkish
- polished level progression
- world/theme transitions
- coins, hints, bonus words, achievements
- PWA-ready web app
- future Android packaging through Capacitor
