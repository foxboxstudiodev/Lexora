# Lexora Privacy Policy Draft

Status: **DRAFT — NOT LEGAL FINAL**

This document is a working privacy-policy draft for Lexora. It must be reviewed and finalized before publishing the app on any store.

## Product

Lexora is a multilingual word-connect crossword puzzle game.

## Current data model

At the current development stage, Lexora stores gameplay progress locally on the user's device using browser/local app storage.

Current locally stored data includes:

- selected language;
- coins;
- completed levels;
- current level;
- settings such as sound, music, and vibration;
- gameplay statistics;
- daily reward state.

## Current collection status

At the current development stage:

- no account system is implemented;
- no server-side user profile is implemented;
- no third-party analytics SDK is implemented;
- no ads SDK is implemented;
- no in-app purchase SDK is implemented.

## Future changes requiring policy update

This privacy policy must be updated before adding any of the following:

- advertising SDK;
- rewarded ads;
- in-app purchases;
- analytics;
- crash reporting;
- cloud save;
- account/login system;
- push notifications;
- multiplayer/social features;
- any third-party SDK that collects device or usage data.

## Release rule

Do not publish this document as final until:

1. the final monetization stack is selected;
2. third-party SDKs are audited;
3. store data-safety answers match the actual app behavior;
4. legal wording is reviewed and approved.
