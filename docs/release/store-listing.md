## Full description draft

Lexora is a polished word-connect crossword puzzle game built for multilingual play.

Connect letters, discover hidden words, complete crossword grids, earn coins, use smart hints, and progress through travel-inspired locations.

Planned core features:

- multilingual word packs;
- clean crossword grids;
- letter wheel gameplay;
- progressive difficulty;
- travel-style world map;
- achievements;
- daily rewards;
- multiple hint types;
- offline-friendly progress storage.
